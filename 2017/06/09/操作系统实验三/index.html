银行家算法的实现

71115115 王子卓 2017年6月2日

一.实验目的

​ 通过实验,加深对多实例资源分配系统中死锁避免方法——银行家算法的理解,掌握 Linux 环境下银行家算法的实现方法,同时巩固利用 Linux PThread API进行共享数据互斥访问和多线程编程的方法。

二.实验内容

  1. 在 Linux 操作系统上,利用 Linux API 编写多线程应用程序实现银行家算法。
  2. 创建 n 个线程来申请或释放资源,只有保证系统安全,才会批准资源申请。
  3. 通过 Linux PThread API 提供的信号量机制,实现共享数据的并发访问。
